# Simulador de Investimentos - Araújo Investimentos

## Descrição do Projeto
Este projeto consiste em uma ferramenta de simulação financeira desenvolvida para auxiliar no planejamento de patrimônio e dividendos. A planilha permite configurar o salário, definir metas de investimento mensal e visualizar projeções de crescimento para até 30 anos.

##  Funcionalidades
- **Configurações Personalizadas:** Cálculo de sugestão de aporte com base no salário.
- **Simulador Mensal:** Cálculo de patrimônio acumulado e rendimento mensal (dividendos).
- **Análise de Cenários:** Projeções detalhadas para prazos de 2, 5, 10, 20 e 30 anos.
- **Sugestão de Carteira:** Divisão percentual por tipo de FII (Papel, Tijolo, Híbrido, etc.) de acordo com o perfil.

##  Aprendizados
Durante o desenvolvimento deste desafio, aprendi a:
1. Utilizar o GitHub para documentação técnica de projetos.
2. Estruturar lógica financeira para cálculos de juros e dividendos.
3. Organizar arquivos e imagens para apresentação de portfólio.
